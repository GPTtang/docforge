# Test Documents

This directory contains regression fixtures for DocForge.

## Generated Office Fixtures

The following Office files are generated by `scripts/generate_test_documents.py` and are safe to regenerate in place:

- `word/docx/unit_test_formatting.docx`
- `word/docx/tablecell.docx`
- `powerpoint/pptx/powerpoint_sample.pptx`
- `powerpoint/pptx/powerpoint_with_image.pptx`
- `excel/xlsx/xlsx_01.xlsx`
- `excel/xlsx/xlsx_05_table_with_title.xlsx`
- `assets/tiny-red.png`

They are designed to exercise:

- headings, inline formatting, lists, tables, and Unicode in `DOCX`
- slide titles, bullets, tables, notes, and images in `PPTX`
- formulas, hyperlinks, comments, merged cells, and multi-sheet layouts in `XLSX`

Regenerate them with:

```bash
./.venv/bin/python scripts/generate_test_documents.py
```

## PDF Fixtures

The PDF files under `pdf/` remain as curated samples for text, OCR, and multi-page behavior.
